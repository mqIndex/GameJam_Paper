新手教学流程（第一天强制触发）
目的：让不玩股票的普通人用最短时间理解“建仓 → 拉升 → 兑现”这个核心闭环。
教学关第一步：认识界面
- 宝叔出现，对话：（立绘加文字）(这是新加界面)
     介绍人物界面布局 参考Project\assets\introduce_baoshu.png
    立绘使用这张Project\assets\baoshu_avatar_UpperHalf.png
- “来得正好，我是你的组长，叫我宝叔就行”
- “下面开始你第一天的交易，我来教你怎么在市场里赚钱”

- 系统用箭头+文字，依次高亮三个核心数值：(目前已实现,先忽略)
  - 资金（左下角）：“这是你的本金，也是你的命根子。”
  - 筹码（左侧）：“你手里的股票数量，货多才能赚大钱。”
  - 市场热度（顶部）：“市场是冷是热，决定了你买卖的效果。”

教学关第二步：建仓（学“买”）(已实现部分功能,修改了文本, 添加了玩家对话,玩家对话框就显示在宝叔对话框下)

宝叔：知道怎么靠股票赚钱吗？

玩家：靠技术分析？

宝叔：
不，靠别人犯傻。
记住，股票本身不值钱。
有人愿意更贵买，它才值钱。

- 宝叔：“先买点货，手里没东西，涨了也跟你没关系。”
- 系统发5张牌，其中保证有一张“买入”，高亮并闪烁。
- 文字提示：点击这张买入
- 玩家打出后：
  筹码增加
  弹幕飘过：“有人进场了！”
  

教学关第二步：拉升（学“造势”）
- 宝叔：“接下来教你市场最核心的东西：情绪”
- 【玩家】：基本面呢？
- 【宝叔】:那是给套牢的人看的。
- 系统发牌，其中保证有一张“大V吹票”，高亮。
- 文字提示：“打出【大V吹票】，拉升市场热度。”
- 玩家打出后：
  - 市场热度条明显上涨
  - 股价加速上涨
  - 弹幕变密集：“要起飞了！”“冲冲冲！”
  - 宝叔：“热度起来了，股价就会自己往上跑，这叫借势。”

教学关第三步：暴力拉升

【玩家】：
等等。
拉升和买入是分开的？
【宝叔】：
废话。
买入是拿筹码，拉升是做气氛，机构有的是手段

- 玩家使用【游资进场】
- 玩家打出后：
  - 股价暴涨
  - 弹幕：
  “满仓干了！”
   “老师我悟了！”
   “技术牛市！”


教学关第四步：兑现（学“卖”）
- 此时股价已到高位，资金 + 筹码市值已接近或超过110,000。
- 宝叔：“现在涨得不错了，但账上的都是虚的，卖掉才是你的钱。来，分批出，别一次砸。”
- 系统发牌中保证有一张“卖出”，高亮。
- 文字提示：“股价已在高位，打出售牌，把筹码换回现金。”
- 玩家打出后：
  - 筹码减少，资金明显增加
  - 弹幕：“有人跑了？”“获利了结很正常”
  - 宝叔：“利好最大的作用，是方便出货。”
  
教学关第五步：尾盘
- 玩家仍剩部分持仓。
- 【宝叔】：
“我们做游资短线，使用的杠杆资金，重要的当天进，当天出，如果当天你出不掉货，我们只能8折贱卖”
【玩家】：为啥啊，我辛辛苦苦拉的股价
【宝叔】：速度出货，是交易的法则

教学关第六步：商店 (这次把商店引导也放在新手关卡吧, 正式关卡的商店也不引导了, 新手关卡购买的卡牌是能够带入正式关的)
【宝叔】：这里是盘后市场，在这里可以补充市场手段，
系统介绍商店基本功能

【宝叔】：虽然市场崇尚交易，但是我们也有内部福利，
高亮“天赋界面”：选择免费的天赋：连携效应
【宝叔】：这能让你的买卖节奏更加顺畅


教学关第七步：事件
刷出一个利空事件，系统介绍
【宝叔】：俗话说，天有不测风云
【玩家】：所以呢
【宝叔】：所以今天开盘大跌，一碗大面，昨天跑的对吧
【玩家】：。。。。
【宝叔】：今天的思路是先打压股价，低位买入，然后拉高出货，记得尾盘跑路

最后进入正式关卡
