二、新手教学关流程（第一局强制触发）
目的：让不玩股票的普通人用最短时间理解“建仓 → 拉升 → 兑现”这个核心闭环。
教学关第一步：认识界面
- 宝叔出现，对话：“小子，想赚钱，先看懂这三样东西——你的钱、你手里的货、还有市场的脸色。”
- 系统用箭头+文字，依次高亮三个核心数值：
  - 资金 ：“这是你的本金，也是你的命根子。”
  - 筹码 ：“你手里的股票数量，货多才能赚大钱。”
  - 市场热度（顶部）：“市场是冷是热，决定了你买卖的效果。”（教学关暂时把上涨/下跌情绪合并为单一热度条）
教学关第二步：建仓（学“买”）
- 宝叔：“先买点货，手里没东西，涨了也跟你没关系。”
- 系统发5张牌，其中保证有一张“做多I”，高亮并闪烁。
- 文字提示：“点击这张【做多I】，用5%的资金买入筹码。”
- 玩家打出后：
  - 资金 -5%，筹码增加
  - 股价小幅上涨
  - 弹幕飘过：“有人进场了！”“涨了涨了！”
  - 宝叔：“看到了吗？你一买，它就涨。”      “买和卖的数量尽量平衡”
教学关第三步：拉升（学“造势”）
- 宝叔：“光自己买不行，得让别人也想买。试试这个。”
- 系统补发2张牌，其中保证有一张“大V吹票”，高亮。
- 文字提示：“打出【大V吹票】，花5%资金拉升市场热度。”
- 玩家打出后：
  - 市场热度条明显上涨
  - 股价加速上涨
  - 弹幕变密集：“要起飞了！”“冲冲冲！”
  - 宝叔：“热度起来了，股价就会自己往上跑，这叫借势。”
教学关第四步：兑现（学“卖”）
- 此时股价已到高位，资金 + 筹码市值已接近或超过110,000。
- 宝叔：“现在涨得不错了，但账上的都是虚的，卖掉才是你的钱。来，分批出，别一次砸。”
- 系统发牌中保证有一张“卖出牌”，高亮。
- 文字提示：“股价已在高位，打出售牌，把筹码换回现金。”
- 玩家打出后：
  - 筹码减少，资金明显增加
  - 股价小幅回落
  - 弹幕：“有人跑了？”“获利了结很正常”
  - 宝叔：“记住这个感觉——落袋为安，贪心的都死得惨。”
教学关第五步：结算
- 资金自动统计，系统判定达标。
- 弹窗：“恭喜！你完成了人生第一笔交易！”
- 宝叔：“悟性不错嘛。以后的路还长，记住今天的三件事：低买 → 造势 → 高位卖。剩下的，自己去市场里摸爬滚打吧。”
- 解锁正式游戏。
